# 🚗 Start App - Sistema de Estacionamento

O **Start App** é um projeto escolar em desenvolvimento focado em facilitar a gestão e reserva de vagas de estacionamento. O objetivo é criar uma experiência fluida tanto para motoristas quanto para administradores de pátios.

---

# 📌 Status do Projeto
O projeto encontra-se em **desenvolvimento ativo**, com atualizações semanais realizadas em ambiente de aula. Atualmente, a fundação do aplicativo (autenticação) está sendo finalizada.

---

# 🚀 Funcionalidades Atuais
Até o momento, o sistema conta com:
* **Tela de Login:** Interface para acesso de usuários.
* **Tela de Cadastro:** Fluxo para novos usuários criarem suas contas.
* **Navegação:** Integração básica entre as telas de autenticação.

---

# 🛠️ Tecnologias Utilizadas
* **IDE:** Android Studio
* **Linguagem:** Kotlin
* **Interface:** XML (View System)
* **SDK Mínima:** API 24 (Android 7.0)
